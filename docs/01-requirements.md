# AI物理教学助手 - 需求分析文档

## 1. 项目背景

### 1.1 目标用户
- **主要用户**：高中物理教师（内部用户，优先服务）
- **次要用户**：高中学生（二期扩展）

### 1.2 核心价值
为物理教师提供一站式 AI 辅助工具，覆盖备课→答疑→出题→资料管理的完整 workflow。

---

## 2. 功能需求

### 2.1 一期：教师端核心功能（MVP）

#### F1: 知识库管理（RAG 基础）
| 需求项 | 描述 |
|--------|------|
| F1.1 文档上传 | 支持 PDF、Word、PPT、图片等格式上传教材、课件、历年考题 |
| F1.2 智能解析 | 提取文本、公式（LaTeX）、表格、图像描述 |
| F1.3 知识检索 | 按知识点、章节、题型等多维度检索 |
| F1.4 文档标注 | 教师可对文档内容进行标记、批注、补充 |

#### F2: AI 备课助手
| 需求项 | 描述 |
|--------|------|
| F2.1 教案生成 | 根据知识点自动生成教学目标、重难点、教学建议 |
| F2.2 资料推荐 | 基于当前备课内容推荐相关例题、实验视频、拓展阅读 |
| F2.3 课件辅助 | 协助整理 PPT 大纲，生成图表/示意图代码 |

#### F3: AI 答疑助手
| 需求项 | 描述 |
|--------|------|
| F3.1 教师答疑 | 教师输入学生问题，AI 提供多种解法的详细分析 |
| F3.2 公式渲染 | 所有物理公式使用 LaTeX 正确渲染 |
| F3.3 分步推理 | 复杂问题展示清晰的解题步骤和物理原理 |

#### F4: AI 出卷助手
| 需求项 | 描述 |
|--------|------|
| F4.1 智能组卷 | 根据知识点、难度、题型自动组卷 |
| F4.2 变式生成 | 基于现有题目生成相似变式题 |
| F4.3 答案详解 | 自动生成标准答案和详细解析 |
| F4.4 LaTeX 导出 | 试卷导出为可编译的 LaTeX 格式 |

#### F5: 可视化与多媒体
| 需求项 | 描述 |
|--------|------|
| F5.1 物理现象图表 | AI 生成 matplotlib/TikZ 代码绘制物理示意图 |
| F5.2 视频资源链接 | 关联优质物理教学视频（外部/自建库）|
| F5.3 实验模拟 | 简单物理仿真（如运动学、力学）交互演示 |

## 3. 非功能需求

### 3.1 技术要求
- **并发**：支持教师日常使用，无需高并发（先单机/小范围）
- **响应时间**：常规问答 <3s，复杂生成 <10s
- **准确率**：物理概念准确率 >95%，需教师审核机制

### 3.2 用户体验
- LaTeX 公式实时渲染
- Markdown 支持的富文本编辑
- 拖拽上传、批量处理
- 对话历史保存和回溯

### 3.3 安全与隐私
- 教学资料本地存储（敏感内容不外泄）
- 支持离线使用（LLM API 可选本地模型替代）

---

## 4. 内容输入清单

### 4.1 输入类型
1. **教材**：人教版高中物理必修+选修电子版
2. **课件**：PPT、PDF 格式的教学课件
3. **题库**：历年高考真题、模拟题、单元测试题
4. **实验视频**：演示实验、虚拟实验资源
5. **参考资料**：大学物理教材、科普文章

### 4.2 格式支持优先级
| 优先级 | 格式 | 处理方式 |
|--------|------|----------|
| P0 | PDF | PyMuPDF / pdfplumber |
| P0 | Word (.docx) | python-docx |
| P0 | 图片 | OCR (PaddleOCR / Tesseract) |
| P1 | PPT (.pptx) | python-pptx |
| P1 | Excel (.xlsx) | openpyxl |
| P2 | 网页 | 爬虫 / reader-link |

---

## 5. 技术选型约束

按照转型学习路线要求：

- ✅ Python 工程能力（异步、Pydantic、类型注解）
- ✅ LLM API 熟练调用（通义/智谱/DeepSeek，支持 Function Calling）
- ✅ LangChain 框架（LCEL、RAG Chain、Agent）
- ✅ 完整 RAG 链路（文档→Embedding→向量库→检索→生成）
- ✅ 结构化输出（Pydantic Schema）

**暂不涉及**：模型微调、CUDA、分布式训练
